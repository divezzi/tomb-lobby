# A/B Test Comparison Notes

## Variant A (variant-a.html)
**Original index.html - Complex particle system**

### Features:
- Interactive particle system with mouse repulsion
- Network effect (connecting lines between particles)
- Gold & cyan particles
- Scanlines overlay
- Tech corner accents on button
- Status indicator with green dot
- "Enter Hub" button

### Performance:
- Heavy JavaScript (particle physics)
- More CPU intensive
- Complex canvas rendering

## Variant B (variant-b.html)
**Original index2.html - Lightweight dust particles**

### Features:
- Simple floating dust/fog particles (90 particles)
- Glitch effect on title (cyan & magenta)
- Vignette overlay
- Cleaner, minimal design
- "ENTER LOBBY" button
- Google Fonts (Cinzel, Roboto)

### Performance:
- Lightweight JavaScript
- Less CPU intensive
- Simple canvas rendering
